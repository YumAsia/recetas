# Recetas Sakura y Más

Widgets de recetas interactivos para [sakuraymas.blogspot.com](https://sakuraymas.blogspot.com/)

## Estructura del repositorio

```
/
├── lentejas-widget.html     ← Widget receta lentejas (selector método + raciones)
├── blogger-iframe.html      ← Código iframe para pegar en Blogger
└── README.md
```

## Cómo usar en Blogger

1. Activar **GitHub Pages** en el repo: Settings → Pages → Branch: main → / (root)
2. La URL del widget será: `https://yumaasia.github.io/recetas/lentejas-widget.html`
3. En la entrada de Blogger → editor → Vista HTML → pegar el contenido de `blogger-iframe.html`

## Características del widget de lentejas

- **5 métodos:** Sakura, Panda, Fuji IH, Vitrocerámica, Thermomix
- **6 opciones de raciones:** 1 / 2 / 3 / 4 / 6 / 8
- Ingredientes escalados automáticamente
- Tabla de tiempos, pasos detallados y consejos PRO por método
- Aviso de capacidad máxima por aparato
- Enlace al blog e Instagram en el footer
